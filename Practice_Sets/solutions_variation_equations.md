# Practice Problems Solutions - Variation Equations  
**Complete Answer Key (Algebra 1 Friendly – No Negative Exponents)**

### Direct Variation (1–6)
1. y = kx → 15 = k·5 → k = 3  
   y = 3x → when x = 12, y = 3·12 = **36**

2. C = kp → 28 = k·8 → k = 3.5  
   C = 3.5p → 15 pounds cost 3.5·15 = **$52.50**

3. d = kt → 180 = k·3 → k = 60  
   d = 60t → in 7 hours: 60·7 = **420 miles**

4. P = kh → 117 = k·9 → k = 13  
   P = 13h → 40 hours: 13·40 = **$520**

5. F = km → 48 = k·12 → k = 4  
   F = 4m → 20 kg needs 4·20 = **80 N**

6. L = kW → 4 = k·10 → k = 0.4  
   L = 0.4W → 25 N stretches 0.4·25 = **10 cm**

### Inverse Variation (7–12)
7. xy = k → 6·20 = 120 → xy = 120  
   When x = 10, y = 120/10 = **12**

8. ts = k → 5·60 = 300 → ts = 300  
   At 80 km/h: t = 300/80 = **3.css3.75 hours** (3 h 45 min)

9. tp = k → 15·6 = 90 → tp = 90  
   For t = 10 h: p = 90/10 = **9 painters**

10. PV = k → 80·30 = 2400 → PV = 2400  
    When V = 50, P = 2400/50 = **48**

11. dw = k → 20·12 = 240 → dw = 240  
    With 30 workers: d = 240/30 = **8 days**

12. I = k/d² → 100 = k/3² → 100 = k/9 → k = 900  
    When d = 6 m: I = 900/36 = **25**

### Joint Variation (13–18)
13. z = kxy → 24 = k·3·4 → k = 2  
    z = 2xy → 2·5·6 = **60**

14. V = klwh → 24 = k·2·3·4 → k = 1  
    Volume is just length × width × height → 5·5·2 = **50 m³**

15. C = kgh → 360 = k·20·3 → k = 6  
    C = 6gh → 6·30·5 = **$900**

16. P = kIR → 48 = k·4·3 → k = 4  
    P = 4IR → 4·6·5 = **120**

17. A = kbh → 60 = k·10·12 → k = 0.5 (same as ½ base·height)  
    A = 0.5·15·8 = **60**

18. F = k·m₁m₂/r → 20 = k·5·8/4 → 20 = k·10 → k = 2  
    F = 2·m₁m₂/r → 2·10·6/3 = 120/3 = **40**

### Combined & Power Variation (19–25)
19. y = k·x/w → 18 = k·9/3 → 18 = k·3 → k = 6  
    y = 6·x/w → 6·20/5 = **24**

20. t = k·d/s → 5 = k·300/60 → 5 = k·5 → k = 1  
    t = d/s → 300/75 = **4 hours**

21. F = k·d/t² → 50 = k·10/2² = 10k/4 → k = 20  
    F = 20·d/t² → 20·15/3² = 300/9 = **33.33 N** (or 100/3 N)

22. W = k·w·h²/L  
    600 = k·4·8²/200 = k·4·64/200 = k·256/200 → k = 600·200/256 = 468.75  
    New beam: W = 468.75·5·10²/250 = 468.75·500/250 = 468.75·2 = **937.5 kg**

23. KE = k·m·v² → 200 = k·4·5² = k·4·25 = 100k → k = 2  
    KE = 2·m·v² → 2·6·10² = 12·100 = **1200 units**

24. R = k·L/d² → 8 = k·100/2² = 100k/4 → k = 8·4/100 = 0.32  
    New wire: R = 0.32·150/4² = 48/16 = **3 ohms**

25. ★ CHALLENGE ★  
    t = k·V/r² → 8 = k·1200/3² = 1200k/9 → k = 8·9/1200 = 0.06  
    New tank: t = 0.06·2000/4² = 120/16 = **7.5 minutes**
